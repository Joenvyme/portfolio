# Joenvyme — Portfolio

Personal one-page portfolio of **Joey Montani** (Joenvyme), creative developer & digital creator — visuals, software and digital art at the crossroads of design, code and AI.

Dark, "negative" identity: near-black background, light type, a giant centered wordmark with the 3D avatar layered on top so the name reads behind the head.

## Stack

- [Next.js 14](https://nextjs.org/) (App Router)
- [Tailwind CSS](https://tailwindcss.com/)
- Type: [Syne](https://fonts.google.com/specimen/Syne) (display), [Space Mono](https://fonts.google.com/specimen/Space+Mono) (accents), Inter (body) via `next/font`

## Sections

- **Hero** — wordmark + 3D avatar, role & tagline
- **Identity** — manifesto, skills marquee, stats
- **Work** — selected projects, each row links straight to the live site
- **References** — testimonials
- **Contact** — email & links

## Development

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build
npm run start    # serve the production build
```
